Wheel Speed Sensor - Rear - Left Side - 4 Cylinder

Print

Operation Code: 17.06.36-02

    Removal

  1. Remove rear wheel. Refer to procedure.
  2. Disconnect wiring harness from harness mounting clip and wheel speed sensor.
  3. Remove M6x16 bolt securing wheel speed sensor to hub carrier. Torque 8 Nm.
  4. Remove wheel speed sensor.

  2. After installation perform a diagnostic read and clear error memory using Lotus Insight Tool.

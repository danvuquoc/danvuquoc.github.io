Coolant Return Line - 4 Cylinder

Print

Operation Code: 40.21.22-02

    Removal

  1. Drain coolant. Refer to procedure.
  2. Remove engine cover. Refer to procedure.
  3. Remove bolt (2) securing coolant return line (1) to turbocharger. Torque 8 Nm.
  4. Remove bolt (3) securing coolant return line to turbocharger. Torque 8 Nm.
  5. Remove bolt (6) securing coolant return line to oil return line. Torque 8 Nm.
  6. Release clip (5) securing oxygen sensor harness to coolant return line.
  7. Release clamp (4) securing coolant return line to engine.

    8. NOTE: Discard clamp.

  8. Remove coolant return line.

    9. NOTE: Be prepared to catch any excess fluid.

    NOTE: Plug exposed connections to prevent ingress of dirt.

    Installation

  1. Installation is the reverse of removal procedure except for the following:
  2. Renew discarded clamp.
  3. Refill coolant.

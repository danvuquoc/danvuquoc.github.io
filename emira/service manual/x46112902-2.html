Hose - Coolant Reservoir To Water Pump Pipe - 4 Cylinder

Print

Operation Code: 46.11.29-02

    Removal

  1. Remove airbox assembly. Refer to procedure.
  2. Remove rear undertray. Refer to procedure.
  3. Release clamp securing coolant hose to water pump pipe.
  4. Disconnect coolant hose from water pump pipe.
  5. Allow coolant to drain.

    6. WARNING: Coolant is a toxic substance and can be lethal if ingested.

    NOTE: Drain coolant into a suitable container and dispose of appropriately.

  6. Release clamp securing coolant hose to coolant reservoir.
  7. Disconnect coolant hose from coolant reservoir.
  8. Release clip securing coolant hose to fuel pipe.
  9. Remove coolant hose.

    Installation

  1. Installation is the reverse of removal procedure except for the following:
  2. Refill coolant.
  3. Start engine and allow to idle.

    4. NOTE: Run the engine with the reservoir cap off to bleed the system.

  4. Refit coolant reservoir cap.

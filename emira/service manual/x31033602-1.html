Hub Carrier Spherical Joint - Left Side - Rear

Print

Operation Code: 31.03.36-02

    Removal

  1. Remove hub carrier. Refer to procedure.
  2. Remove old bush using hydraulic press.

    Installation

  1. Installation is the reverse of removal except for the following:
  2. Clean hub carrier bore before installing new bush.
  3. Install spherical bush with rolled edge pushing into hub carrier.
  4. Spherical bush front side (rolled edge) should protrude 13.5mm ± 0.5mm.

Transmission Control Unit (TCU) - V6

Print

Operation Code: 17.09.01-02

    Removal

  1. Disconnect battery. Refer to procedure.
  2. Remove rear wheelarch liner left side. Refer to procedure.
  3. Disconnect harness connectors (x2) from TCU.
  4. Remove M6 nuts (x3) securing TCU to studs. Torque 4.5 Nm.
  5. Remove TCU.

    Installation

  1. Installation is the reverse of removal procedure.
  2. After installation download software using Emira DS.
  3. Perform the gear adaption mode routine using Emira DS.
  4. Perform a diagnostic read and clear error memory using Lotus Insight tool.

Battery Backup Sounder

Print

Operation Code: 17.33.19-02

    Removal

  1. Remove front undertray. Refer to procedure.
  2. Disconnect harness connector from battery backup sounder.
  3. Remove M6x16 bolts (x2) securing battery backup sounder to chassis. Torque 6 Nm.
  4. Remove battery backup sounder.

    5. Do not carry out further disassembly if component is removed for access only.

  5. Loosen M6 nuts (x2) and remove brackets (x2) from battery backup sounder. Torque 6 Nm.

    Installation

  1. Installation is the reverse of removal procedure except for the following:
  2. After installation perform a diagnostic read and clear error memory using Lotus Insight tool.

    3. NOTE: Register module using Lotus Insight tool.

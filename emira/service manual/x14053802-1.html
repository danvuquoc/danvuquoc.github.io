Sun Sensor / Speaker Panel

Print

Operation Code: 14.05.38-02

    Removal

  1. Use suitable flat bladed trim removal tool to carefully disengage clips (x10) to release sun sensor/speaker panel from facia.
  2. Refer to illustration for clip type and location.
  3. Lift sun sensor/speaker panel to gain access to sun sensor harness connector and disconnect.
  4. Remove sun sensor/speaker panel from vehicle.

    5. Do not carry out further disassembly if component is removed for access only.

  5. Disengage clips (x3) securing sun sensor to speaker panel.
  6. Remove sun sensor from speaker panel.

    Installation

  1. Installation is the reverse of removal procedure except for the following:
  2. Perform a diagnostic read and clear error memory using Lotus Insight tool.

Actuator - Recirculation Flaps - RHD

Print

Operation Code: 18.01.10-02

    Removal

  1. Remove heater / air conditioning housing assembly. Refer to procedure.
  2. Disconnect harness connector from left side actuator.
  3. Remove self tapping screws (x3) securing left side actuator to HVAC unit.
  4. Remove left side actuator.

    Installation

  1. Installation is the reverse of removal procedure.
  2. After installation perform a diagnostic read and clear error memory using Lotus Insight tool.

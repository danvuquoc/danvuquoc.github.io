Tailgate Release Cable - Latch to Actuator

Print

Operation Code: 10.11.23-02

    Removal

  1. Remove tailgate latch cover. Refer to procedure.
  2. Release straps (x2) and remove vehicle tool kit.
  3. Release strap and remove warning triangle.
  4. Remove luggage floor carpet.
  5. Release clips (x8) securing rear boot trim to boot box.
  6. Refer to illustration for trim clip location.
  7. Remove rear boot trim.
  8. Disengage tailgate release cable from tailgate latch actuator.
  9. Disconnect tailgate handle from tailgate release cable.
  10. Release clips (x2) and open tailgate release cable cover.
  11. Disconnect tailgate manual release cable from tailgate release cable.
  12. Release clips (x4) securing tailgate release cable to boot box
  13. Remove tailgate release cable from bracket.

    Installation

  1. Installation is the reverse of removal procedure.

Harness - Engine - 4 Cylinder

Print

Operation Code: 17.01.02-02

    Removal

  1. Disconnect battery. Refer to procedure.
  2. Remove engine cover. Refer to procedure.
  3. Remove rear wheelarch liner - left side. Refer to procedure.
  4. Remove rear undertray. Refer to procedure.
  5. Remove rear bulkhead access panel. Refer to procedure.
  6. Disconnect harness connectors (x2) from airbox and engine bay harness.
  7. Disconnect harness connectors (x6) from ECM.
  8. Disconnect harness connector from engine bay harness.
  9. Cut cable ties (x3) securing engine harness to airbox.
  10. Disconnect harness connectors (x2) from solenoid and purge valve.
  11. Release clips (x2) securing harness to engine.
  12. Disconnect harness connectors (x5) from solenoid, pressure sensor, temperature sensor, DPS module and inlet hose.
  13. Release clips (x2) securing harness to engine.
  14. Remove screw securing harness to engine.
  15. Disconnect harness connector from oxygen sensor.
  16. Release clip securing harness to engine.
  17. Disconnect harness connectors (x4) from oil level sensor, AC compressor, crankshaft sensor and water pump.
  18. Release clips (x5) securing harness to engine.
  19. Disconnect harness connector from throttle body.
  20. Release clip securing harness to engine.
  21. Release clips (x3) securing harness to engine.
  22. Disconnect harness connectors (x2) from alternator and hydraulic valve.
  23. Release clip securing harness to engine.
  24. Disconnect harness connectors (x4) from temperature sensor, oxygen sensor, fuel rail sensor and charge air distributor.
  25. Release clips (x2) securing harness to engine.
  26. Disconnect harness connectors (x4) from fuel injectors.
  27. Release clips (x3) securing harness to engine.
  28. Release harness connectors (x3) from camshaft sensors (x2) and catalytic converter temperature sensor.
  29. Release clip securing harness to engine.
  30. Disconnect harness connectors (x4) from ignition coils.
  31. Disconnect harness connectors (x4) from injectors.
  32. Disconnect harness connectors (x2) from lift solenoids.
  33. Release hoses (x3) from engine harness.
  34. Release clip securing hose to engine harness.
  35. Remove screws (x3) securing engine harness to engine. Torque 8 Nm.
  36. Remove engine harness.

    Installation

  1. Installation is the reverse of removal procedure except for the following:
  2. Renew cable ties.

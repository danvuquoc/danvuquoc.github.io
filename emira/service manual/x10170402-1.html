Door Cable - Interior Release

Print

Operation Code: 10.17.04-02

    Removal

  1. Remove door latch module left side. Refer to procedure.
  2. Remove self tapping screws (x6) securing door latch module top cover to door latch module body.
  3. Disengage clips (x2) securing door latch module top cover to door latch module body.
  4. Remove door latch module top cover.
  5. Disengage clip securing interior door release cable to door latch module.
  6. Remove interior door release cable.

    Installation

  1. Installation is the reverse of removal procedure.

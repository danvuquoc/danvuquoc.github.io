Crankshaft Position Sensor - V6

Print

Operation Code: 17.06.03-02

    Removal

  1. Remove air conditioning compressor. Refer to procedure.
  2. Disconnect harness connector from crankshaft position sensor.
  3. Remove bolt securing crankshaft position sensor to engine. Torque 10 Nm.
  4. Remove crankshaft position sensor.

    Installation

  1. Installation is the reverse of removal procedure except for the following:
  2. Apply new engine oil to O-ring before installation.

Parking Brake Pads

Print

Operation Code: 33.01.13-02

    Removal

  1. Remove rear wheel. Refer to procedure.
  2. Set parking brake caliper to service position using Lotus insight tool.
  3. Remove brake pad retaining clips (x2).
  4. Remove the brake pad retaining pins (x2).
  5. Remove brake pad anti-rattle spring.
  6. Remove brake pads.

    Installation

  1. Installation is the reverse of removal procedure except for the following:
  2. Reset caliper from service position using Lotus insight tool.

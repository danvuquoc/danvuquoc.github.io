Door Latch Striker - Left Side

Print

Operation Code: 10.17.23-02

    Removal

  1. Open front door - left side.
  2. Remove M8x35 bolts (x2) securing door latch striker to B pillar. Torque 30 Nm.

    3. NOTE: To aid installation mark position of door latch striker to B pillar.

  3. Remove the door latch striker and seal from the vehicle.

    Installation

  1. Installation is the reverse of removal procedure except the following:
  2. Install door latch in the correct position. Use position marked during removal procedure.
  3. If door latch striker requires shim adjustment perform the following procedure:
  4. Remove M5x20 bolt securing spacer and shims to B pillar. Torque 6 Nm.
  5. Remove spacer, shim locator and shims from B pillar.
  6. Remove or add shims as required.

    7. NOTE: Always record quantity and fitted position of shims.

  7. Install shim assembly to B pillar then continue with door latch striker installation.
  8. Check door opens and closes smoothly without excessive contact with door latch striker.

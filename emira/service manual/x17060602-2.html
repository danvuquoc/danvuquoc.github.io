Oxygen Sensor - Post Catalytic Converter - 4 Cylinder

Print

Operation Code: 17.06.06-02

    Removal

  1. Remove engine cover. Refer to procedure.
  2. Remove bootbox access panel. Refer to procedure.
  3. Disconnect harness connector from oxygen sensor.
  4. Release clip securing oxygen sensor connector to bracket.
  5. Release clip securing oxygen sensor to bracket.
  6. Unscrew oxygen sensor from catalytic converter. Torque 45 Nm.
  7. Remove oxygen sensor.

    Installation

  1. Installation is the reverse of removal procedure except for the following:
  2. After installation perform a diagnostic read and clear error memory using Lotus Insight tool.

Camshaft Position Sensors - V6

Print

Operation Code: 17.06.16-02

    Removal

  1. Remove supercharger assembly. Refer to procedure.
  2. Remove boot box access panel. Refer to procedure.
  3. Disconnect harness connectors (x4) from camshaft position sensors.
  4. Remove bolts (x4) securing camshaft position sensors to engine. Torque 10 Nm.
  5. Remove camshaft position sensors.

  2. After installation perform a diagnostic read and clear error memory using Lotus Insight tool.

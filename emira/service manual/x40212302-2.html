Oil Return Line - 4 Cylinder

Print

Operation Code: 40.21.23-02

Important Information

WARNING: Engine oil is a toxic substance and can be lethal if ingested. Wear protective gloves, protective clothing and safety glasses.

    Removal

  1. Remove engine cover. Refer to procedure.
  2. Remove rear undertray. Refer to procedure.
  3. Remove bolt (2) securing oil return line (1) to turbocharger. Torque 8 Nm.
  4. Remove bolt (3) securing coolant pipe to oil return line. Torque 8 Nm.
  5. Remove bolt (4) securing oil return line to engine. Torque 8 Nm.
  6. Remove oil return line.

    7. NOTE: Be prepared to catch any excess fluid.

    NOTE: Plug exposed connections to prevent ingress of dirt.

    NOTE: Discard O-rings.

    Installation

  1. Installation is the reverse of removal procedure except for the following:
  2. Renew discarded O-rings.
  3. Check oil level and top up if necessary.

Forward Crash Sensor - Left Side

Print

Operation Code: 17.35.15-02

    Removal

  1. Remove front undertray. Refer to procedure.
  2. Disconnect harness connector from forward crash sensor.
  3. Remove M6x25 bolt securing forward crash sensor to bracket. Torque 10 Nm.
  4. Remove forward cash sensor.

    Installation

  1. Installation is the reverse of removal procedure except for the following:
  2. Perform a diagnostic read and clear error memory using Lotus Insight tool.
  3. Turn on ignition and make sure SRS warning lamp extinguishes.

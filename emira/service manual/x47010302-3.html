Clutch Assembly - V6

Print

Operation Code: 47.01.03-02

    Removal

  1. Remove transmission assembly. Refer to procedure.
  2. Disengage clutch fork from clutch release bearing.
  3. Remove clutch fork from transmission.
  4. Remove clutch release bearing from input shaft.
  5. Clean bellhousing and input shaft using suitable degreaser.
  6. Remove M8x16 bolts (x6) securing clutch cover to flywheel. Torque 20 Nm.

    7. CAUTION: Loosen bolts in a diagonal sequence to release the clamp load without distorting the clutch cover.

    NOTE: Apply permabond A130 to bolts during installation.

  7. Match mark clutch cover and flywheel.
  8. Remove clutch cover.
  9. Remove clutch friction disc.

    Installation

  1. Installation is the reverse of removal procedure except for the following:
  2. Use clutch alignment tool to position clutch friction disc on to flywheel.

    3. NOTE: The end of the clutch alignment tool should not protrude further than the clutch cover spring fingers.

  3. Install clutch compression tool T000T1524F.
  4. Operate clutch compression tool until there is no gap between clutch cover and flywheel.
  5. Complete two full turns to preload clutch diaphragm.
  6. Tighten installed clutch cover bolts (x3). Torque 20 Nm.
  7. Remove clutch compression tool T000T1524F.
  8. Install remaining clutch cover bolts (x3). Torque 20 Nm.
  9. Remove clutch alignment tool.
  10. Remove dirt or grease from contact surfaces of clutch fork to release bearing, support peg and slave cylinder push rod.
  11. Apply a small amount of Fuchs RENOLIT PASTE PW to support peg recess.
  12. Apply a small amount of Fuchs RENOLIT PASTE PW to slave cylinder push rod.
  13. Apply small amount of Fuchs RENOLIT PASTE PW to input shaft collar on transmission.
  14. Install clutch fork to transmission.

    15. NOTE: Locate release bearing on input shaft collar whilst passing opposite end of clutch fork through aperture. Align clutch fork recess against support peg, then firmly push clutch fork to fully secure on support peg.

  15. Apply small amount of Fuchs RENOLIT PASTE PW to first 10 mm of input shaft.

    16. CAUTION: Remove any excess grease from chamfer on input shaft which could contaminate clutch assembly causing clutch judder.

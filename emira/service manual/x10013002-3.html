Boot Box - V6

Print

Operation Code: 10.01.30-02

Procedure Introduction

NOTE: View boot box video guide on dealer portal before commencing procedure.

    Removal

  1. Remove rear engine bay trim cover. Refer to procedure.
  2. Remove tailgate. Refer to procedure.
  3. Remove C pillar left side. Refer to procedure.
  4. Remove C pillar right side.

    5. NOTE: Procedure is the same as left side component.

  5. Remove tailgate latch actuator. Refer to procedure.
  6. Remove rear luggage compartment trim. Refer to procedure.
  7. Remove washer bottle assembly. Refer to procedure.
  8. Remove battery. Refer to procedure.
  9. Remove silencer assembly. Refer to procedure.
  10. Disengage clips (x8) securing body harness to boot box.
  11. Remove grommet from boot box.
  12. Pull harness from boot box and remove body harness from boot box.
  13. Disengage clips (x2) securing battery fuse box cover to fuse box.
  14. Remove battery fuse box cover.
  15. Remove M8 nuts (x3) securing battery cables (x3) to fuse box. Torque 24 Nm.
  16. Remove M6 nyloc nut securing battery cable to fuse box. Torque 6 Nm.
  17. Remove fuse box.
  18. Disengage clips (x4) securing battery cables to boot box.
  19. Disengage clips (x4) securing battery cable and harness to boot box.
  20. Disengage clips (x7) securing engine harness to boot box.
  21. Remove grommets (x4) from boot box.
  22. Pull battery cables (x4) and harness out of boot box and set aside.
  23. Disengage clips (x2) securing trunk lid release cable cover to trunk lid release cable box.
  24. Remove trunk lid release cable cover.
  25. Disengage manual tailgate release cable from cable box.
  26. Disengage clips (x3) securing manual tailgate release cable to boot box.
  27. Remove grommet from boot box.
  28. Remove manual tailgate release cable from boot box.
  29. Outline bolts (x2) on each side to aid installation.
  30. Measure 25 mm from each bolt into the recess as shown in the image and mark position.
  31. Using a 4 mm drill bit, drill holes (x4) at the marked positions.
  32. Measure the depth of each hole from the top of the boot box to the subframe for each hole.

    33. NOTE: Take note of the measurement of each hole.

  33. Mark lines from boot box to subframe to aid re-alignment on installation.
  34. Mark lines from boot box to engine bay area to aid re-alignment on installation.
  35. Remove M8x50 bolts (x5) securing battery support plate to boot box. Torque 24 Nm.
  36. Remove bolts (x8) securing boot box to rear subframe. Torque 10 Nm.

    37. NOTE: Make sure the fixings under the boot box remain in the top position and do not move downwards.

  37. Remove M6x25 bolts (x5) securing boot box to rear subframe. Torque 10 Nm
  38. Using assistance remove boot box.

    Installation

  1. Installation is the reverse of removal procedure except for the following:
  2. Seal holes (x4) in bootbox.

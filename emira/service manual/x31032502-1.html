Bush - Lower Wishbone - Rear - Left Side

Print

Operation Code: 31.03.25-02

    Removal

  1. Remove rear lower wishbone left side. Refer to procedure.
  2. Remove old bushes using hydraulic press.
  3. Bushes can only be removed in one direction. Refer to illustration.

    Installation

  1. Installation is the reverse of removal procedure except for the following:
  2. Clean wishbone bores before installing new bushes.
  3. Lubricate bush outer surface using P80 rubber lubricant to ease installation.
  4. Make sure bushes are inserted into side of wishbone bore with chamfered edge.

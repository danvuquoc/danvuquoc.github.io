Fan Speed Control Module - LHD - V6

Print

Operation Code: 18.01.07-02

    Removal

  1. Disconnect battery. Refer to procedure.
  2. Remove front undertray. Refer to procedure.
  3. Remove M4x16 bolts (x2) securing fan speed control module and cover to HVAC unit.
  4. Remove cover from HVAC unit.
  5. If fitted, remove water shield from HVAC unit. Refer to TSB-131-24007.
  6. Disengage harness connectors (x2) from fan speed control module.
  7. Remove fan speed control module.

    Installation

  1. Installation is the reverse of removal procedure except for the following:
  2. After installation perform a diagnostic read and clear memory using Lotus Insight tool.

Sensor - Fuel Pressure - V6

Print

Operation Code: 44.01.08-02

    Removal

  1. Remove fuel tank. Refer to procedure.
  2. Remove M6 nyloc nuts and washers (x6) securing fuel pump locking ring to fuel tank.

    3. CAUTION: Apply a suitable lithium based lubricant such as Castrol Moly Grease to threads of the fuel pump retaining studs to prevent ‘Galling’.

    NOTE: Remove nuts in a diagonal sequence.

  3. Whilst using finger pressure to hold the fuel pump in position, remove fuel pump locking ring.
  4. Remove fuel pressure sensor.

    5. NOTE: Continue using finger pressure to hold fuel pump in position.

    Installation

  1. Install fuel pressure sensor.

    2. NOTE: Continue using finger pressure to hold fuel pump in position.

    NOTE: Remove protective port cap and electrical connector cover from new fuel pressure sensor.

    NOTE: Position fuel pressure sensor so electrical connection is facing inwards to tank and sensor body is parallel to fuel pump retaining studs.

  2. Install fuel pump locking ring.

    3. NOTE: Continue using finger pressure to hold fuel pump in position.

  3. Install new nyloc nuts and washers (x6). Torque 2 Nm.

    4. NOTE: Continue using finger pressure to hold fuel pump in position.

    NOTE: Tighten nuts in a diagonal sequence.

  4. Apply a final tightening to nyloc nuts (x6). Torque 3 Nm.

    5. CAUTION: Ensure clamp ring is flush against top of fuel pump flange.

    NOTE: Tighten nuts in a diagonal sequence.

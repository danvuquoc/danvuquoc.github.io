Controller Unit Assembly - 4 Cylinder

Print

Operation Code: 47.02.16-02

Important Information

WARNING: Transmission oil is a toxic substance and can be lethal if ingested. Wear protective gloves, protective clothing and safety glasses.

    Removal

  1. If controller unit assembly is to be renewed, perform commissioning procedure using Lotus Insight tool.
  2. Remove oil pan. Refer to procedure.
  3. Disconnect harness connector from controller unit assembly.
  4. Remove and discard M5x60 Torx bolts (5) (x15) securing controller unit assembly (Y3/14) to transmission (3). Torque 3 Nm + 60 degrees.

    5. CAUTION: These fasteners have been torqued to a specification where the threads may have been damaged or stretched. Always discard and replace with new on installation.

  5. Remove controller unit assembly.
  6. Remove and discard sealing ring (8).

    7. Do not carry out further disassembly if component is removed for access only.

  7. Remove and discard oil filter (4).

    Installation

  1. Installation is the reverse of removal procedure except for the following:
  2. Renew discarded sealing ring.
  3. Apply transmission oil to sealing ring.
  4. Renew discarded bolts.
  5. Perform a diagnostic read and clear error memory using Lotus Insight tool.

Sensor - Differential Pressure - V6

Print

Operation Code: 17.06.38-02

    Removal

  1. Disconnect battery. Refer to procedure.
  2. Remove rear wheelarch liner right side.

    3. NOTE: Procedure is the same for left side component.

  3. Disconnect harness connector from differential pressure sensor.
  4. Remove M5x16 bolt securing differential pressure sensor to engine. Torque 5 Nm.
  5. Remove differential pressure sensor.

    Installation

  1. Installation is the reverse of removal procedure except for the following:
  2. After installation perform a diagnostic read and clear error memory using Lotus Insight tool.

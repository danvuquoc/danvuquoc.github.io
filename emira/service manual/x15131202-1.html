Speaker Grille - Left Side

Print

Operation Code: 15.13.12-02

    Removal

  1. Remove interior door pull trim - left side. Refer to procedure.
  2. Remove M4x13 screws (x8) securing speaker grille to interior door pull trim.
  3. Release clips (x2) securing speaker grille to interior door pull trim.
  4. Remove speaker grille.

    5. Do not carry out further disassembly if component is removed for access only.

  5. Release clips (x4) securing seat memory switch to speaker grille.
  6. Remove seat memory switch.

    Installation

  1. Installation is the reverse of removal procedure.

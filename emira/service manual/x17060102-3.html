Engine Coolant Temperature Sensor - V6

Print

Operation Code: 17.06.01-02

    Removal

  1. Allow engine to cool completely.
  2. Disconnect electrical connector from engine coolant temperature sensor.
  3. Use SST 09817-33190 to remove engine coolant temperature sensor. Torque 19.6 Nm.

    4. NOTE: Be prepared to catch any excess fluid.

    Installation

  1. Installation is the reverse of removal procedure except for the following:
  2. After installation perform a diagnostic read and clear error memory using Lotus Insight tool.

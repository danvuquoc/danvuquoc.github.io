Coolant Pipe - 4 Cylinder

Print

Operation Code: 40.21.21-02

    Removal

  1. Drain coolant. Refer to procedure.
  2. Remove engine cover. Refer to procedure.
  3. Remove bolts (x2) securing coolant pipe to engine. Torque 8 Nm.
  4. Release clamps (x2) securing coolant pipe to coolant feed and return lines.

    5. NOTE: Discard clamps.

  5. Remove coolant pipe.

    6. NOTE: Be prepared to catch any excess fluid.

    NOTE: Plug exposed connections to prevent ingress of dirt.

    Installation

  1. Installation is the reverse of removal procedure except for the following:
  2. Renew discarded clamps.

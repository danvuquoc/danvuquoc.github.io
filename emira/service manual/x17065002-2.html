Oil Level Sensor - 4 Cylinder

Print

Operation Code: 17.06.50-02

    Removal

  1. Drain engine oil. Refer to procedure.
  2. Disconnect harness from oil level sensor.
  3. Remove bolts (x3) securing oil level sensor to engine. Torque 8 Nm.
  4. Remove oil level sensor.

    5. WARNING: Engine oil is a toxic substance and can be lethal if ingested.

    NOTE: Catch any excess oil in a suitable container and dispose of appropriately.

  5. Remove and discard O-ring.

    Installation

  1. Installation is the reverse of removal procedure except for the following:
  2. Renew O-ring.
  3. After installation perform a diagnostic read and clear error memory using Lotus Insight tool.

Tailgate Glass

Print

Operation Code: 11.01.11-02

    Removal

  1. Remove tailgate. Refer to procedure.
  2. Release adhesive securing tailgate glass trims (x2) to tailgate.

    3. NOTE: The tailgate glass trim is retained using adhesive strips.

  3. Remove tailgate glass trims (x2).
  4. Using suitable equipment, cut adhesive securing tailgate glass to tailgate.
  5. Lift tailgate glass to access heated screen connectors.
  6. Disconnect tailgate heated screen from tailgate.

    7. NOTE: Reconnect heated screen connectors before bonding glass to tailgate.

  7. Using assistance remove tailgate glass.

    8. NOTE: If glass is to be reused place on suitable surface to prevent damage.

  8. Remove rubber spacers (x8) from tailgate.

    9. NOTE: Always record quantity and fitted position of spacers.

    Installation

  1. Installation is the reverse of removal procedure except for the following:
  2. Install tailgate.

    3. CAUTION: Ensure tailgate is correctly aligned before proceeding.

  3. Inspect heated screen trace lines for damage.
  4. Remove old adhesive from tailgate.
  5. Clean tailgate mating surface using betaclean 3300.
  6. Apply betaprime 5404 primer to tailgate aperture.
  7. Apply betaseal 1580 to tailgate aperture.
  8. Install rubber spacers (x8) into adhesive.
  9. Clean area of glass around tailgate glass heating element using betaclean 3300.
  10. Using assistance install tailgate glass to tailgate.

    11. NOTE: Reconnect heated screen connectors before bonding glass to tailgate.

Drivers Stalk Module

Print

Operation Code: 17.35.03-02

    Removal

  1. Remove lower steering column shroud. Refer to procedure.
  2. Remove upper steering column shroud. Refer to procedure.
  3. Remove steering wheel. Refer to procedure.
  4. Remove clip securing harness to stalk module.
  5. Loosen bolt securing stalk module to upper steering column until stalk module becomes loose.
  6. Release clip locating stalk module to upper steering column.
  7. Remove stalk module from upper steering column.

    Installation

  1. Installation is the reverse of removal procedure except for the following:
  2. Push the stalk module on to the upper steering column until it stops and the clip locates to upper steering column.

    3. NOTE: Make sure the airbag rotary contact is centralised before refitting stalk module.

Valve - Switchover - Coolant - 4 Cylinder

Print

Operation Code: 40.15.30-02

Important Information

WARNING: Risk of scalding. Allow coolant to cool completely before proceeding.

WARNING: Coolant is a toxic substance and can be lethal if ingested.

    Removal

  1. Drain coolant. Refer to procedure.
  2. Remove rear undertray. Refer to procedure.
  3. Disconnect harness connector from coolant switchover valve.
  4. Remove bolts (x2) securing coolant switchover valve to engine. Torque 8 Nm.
  5. Release spring clamps (x3) securing coolant hoses to coolant switchover valve.
  6. Disconnect coolant hoses from coolant switchover valve.

    7. NOTE: Be prepared to catch any excess fluid.

  7. Remove coolant switchover valve.

    Installation

  1. Installation is the reverse of removal procedure except for the following:
  2. After installation perform a diagnostic read and clear error memory using Lotus Insight tool.

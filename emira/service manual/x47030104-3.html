Transmission Assembly Overhaul - V6

Print

Operation Code: 47.03.01-04

    Removal

  1. Remove transmission assembly. Refer to procedure.

    Remove Clutch Release Fork Sub-Assembly

  1. Remove the clutch release fork with clutch release bearing from the manual transaxle case.

    Remove Clutch Release Bearing Bearing Assembly

  1. Remove the clutch release bearing assembly and clutch release bearing hub from the release fork sub-assembly.

    Remove Release Fork Support

  1. Remove the release fork support from the manual transaxle case.

    Remove Clutch Release Fork Boot

  1. Remove the clutch release fork boot from the manual transaxle case.

    Remove Floor Shift Control Lever Housing Support Bracket

  1. Remove the 3 bolts floor shift control lever housing support bracket from the manual transaxle assembly.

    Remove Selecting Bell Crank Assembly

  1. Remove the 2 bolts and selecting bell crank assembly from the manual transmission case.

    Remove Back up Light Switch Assembly

  1. Using SST 09816-30010, remove the back-up light switch assembly and gasket.

    Remove no.1 Lock Ball Assembly

  1. Remove the No.1 lock ball assembly from the manual transmission case.

    Remove no.2 Lock Ball Assembly

  1. Remove the No.2 lock ball assembly from the manual transmission case.

    Remove Shift Gate Pin

  1. Remove the shift gate pin from the manual transmission case.

    Remove Control shaft Cover

  1. Remove the 4 bolts and control shaft cover from the manual transmission case.

    Remove Shift and Select Pin

  1. While pressing the shift and select pin (*1) in as far as it will go, pull the shift and select pin (*2).
  2. Remove the spring.

    Remove O-Ring

  1. Remove the O-ring from the shift and select pin.

    Remove Check Valve

  1. Remove the check valve and gasket from the manual transmission case.

    Remove Temperature Sensor

  1. Remove the temperature sensor and gasket from the front transaxle case.

    Remove Ball

  1. Using a hexagon wrench, remove the 4 shift detent ball plugs from the front transaxle case.
  2. Using a magnetic finger, remove the 4 spring seats and balls.

    Remove Reverse Idler Gear Shaft Bolt

  1. Remove the reverse idler gear shaft bolt and reverse idler gear shaft gasket from the manual transmission case.

    Remove Manual Transmission Case Plug

  1. Remove the manual transmission case plugs from the manual transmission case.

    Remove Manual Transmission Case

  1. Remove the 6 bolts from the front transaxle case side.
  2. Remove the 12 bolts from the manual transmission case side.
  3. Use 2 snap ring pliers to keep the snap ring stretched and remove the manual transmission case.

    Remove no.1 Oil Receiver Pipe

  1. Remove the No.1 receiver pipe from the manual transmission case.

    2. CAUTION: Do not damage the No.1 oil receiver pipe.

    Remove Transmission Oil Separator

  1. Remove the 2 bolts and transmission oil separator from the manual transmission case.

    Remove 1st and 2nd Shift Fork Shaft

  1. Remove the bolt of the 1st and 2nd shift fork shaft.
  2. Remove the 1st and 2nd shift fork shaft.

    Remove 3rd and 4th Shift fork Shaft

  1. Using a pin punch and hammer, remove the slotted spring pin from the No.2 gear shift head.
  2. Using a pin punch and hammer, remove the slotted spring pin from the No.2 gear shaft fork.
  3. Remove the 3rd and 4th shift fork shaft and No.2 gear shift head.

    Remove no.2 Gear Shift Fork

  1. Remove the No.2 gear shift fork from the No.1 output shaft assembly.

    Remove no.5 Gear Shift Fork Shaft

  1. Using a screwdriver, remove the E-ring from the front transaxle case.
  2. Remove the pin from the front transaxle case.
  3. Remove the No.5 shift fork shaft from the front transaxle case.
  4. Using a pin punch and hammer, remove the slotted spring pin.
  5. Remove the No.3 gear shift head.

    Remove no.1 Gear Shift Fork

  1. Remove the No.1 gear shift fork from the No.1 output shaft assembly.

    Remove Shift Arm

  1. Remove the shift arm from the front transaxle case.

    Remove Reverse Shift Fork Shaft

  1. Remove the bolt of the reverse shift fork shaft.
  2. Remove the reverse shift fork shaft.

    Remove 5th and 6th Shift Fork Shaft

  1. Remove the bolt of the No.3 gear shift fork.
  2. Remove the 5th and 6th shift fork shaft.

    Remove no.3 Shift Fork

  1. Remove the No.3 gear shift fork from the No.2 output shaft assembly.

    Inspect Reverse Idler Gear Thrust Clearance

  1. Using a feeler gauge, measure the reverse idler gear thrust clearance.

    2. NOTE: Standard clearance: 0.40 to 1.05 mm (0.0016 to 0.0413 in.).

    NOTE: Maximum clearance: 1.05 mm (0.0413 in.).

  2. If the clearance exceeds the maximum, replace the reverse idler gear, needle roller bearing and reverse idler gear shaft.

    Inspect Reverse Idler Gear Radial Clearance

  1. Using a dial indicator, measure the reverse idler gear radial clearance.

    2. NOTE: Standard clearance: 0.015 to 0.048 mm (0.0006 to 0.0019 in.).

    NOTE: Maximum clearance: 0.048 mm (0.0019 in.).

  2. If the clearance exceeds the maximum, replace the reverse idler gear, needle roller bearing and reverse idler gear shaft.

    Remove Reverse Idler Gear

  1. Remove the reverse idler gear shaft by sliding and lifting it.
  2. Remove the reverse idler gear, needle roller bearing and reverse idler thrust washer from the reverse idler gear shaft.

    Remove Input Shaft, no.1 Output Shaft and no.2 Output Shaft

  1. Remove the 3 shafts at the same time.

    Remove Transmission Oil Separator

  1. Remove the 2 bolts and transmission oil separator from the front transaxle case.

    Remove Front Differential Case Assembly

  1. Remove the differential case assembly from the front transaxle case.

    Remove Transmission Magnet

  1. Remove the transmission magnet from the front trasaxle case.

    Remove no.2 Output Shaft Rear Bearing Outer Race

  1. Using SST 09387-00041 (09387-02020), remove the No.2 output shaft front bearing outer race from the manual transmission case.

    Remove Output Shaft Rear Bearing Shim

  1. Remove the output shaft rear bearing shim from the manual transmission case.

    Remove Shift Fork Shaft Bearing

  1. Using SST 09612-65014 09387-00041 (09387-01021) 09527-20011, remove the 4 shift fork shaft bearings from the manual transmission case.

    Remove Shift and Select Lever Shaft Needle Roller Bearing

  1. Using SST 09612-65014 09387-00041 (09387-01021), remove the shift and select lever shaft needle roller bearing from the manual transmission case.

    Remove Input Shaft Rear Bearing Shaft Snap Ring

  1. Using snap ring pliers, remove the 2 snap rings.

    Remove Input Shaft Front Bearing

  1. Remove the bolt from the front transaxle case.
  2. Using SST 09612-30012, remove the input shaft bearing from the front transaxle case.

    Remove Transaxle Case Oil Seal

  1. Using a screwdriver, remove the transaxle case oil seal from the front transaxle case.

    Remove output Shaft Front Bearing

  1. Remove the bolt from the front transaxle case.
  2. Using SST 09612-65014 (09612-01020), Remove the output shaft front bearing from the front transaxle case.

    Remove Output Shaft Cover

  1. Remove the output shaft cover from the front transaxle case.

    Remove no.2 Output Shaft Front Bearing Outer Race

  1. Using SST 09387-00041 (09387-02020), remove the No.2 output shaft front bearing outer race from the front transaxle case.

    Remove Output shaft Cover

  1. Remove the output shaft cover from the front transaxle case.

    Remove Shift Fork Shaft Bearing

  1. Using SST 09612-65014 09387-00041 (09487-01021), remove the 4 shift fork shaft bearings from the front transaxle case.

    Remove Transmission Case Oil Seal

  1. Using a screwdriver, remove the transmission case oil seal from the manual transmission case.

    Remove Front Differential Case Rear Outer Race Bearing

  1. Using a brass bar and a hammer, remove the front differential case rear outer race bearing and front differential case rear shim from the manual transmission case.

    Remove Transaxle Case Oil Seal

  1. Using a screwdriver, remove the transaxle case oil seal from the front transaxle case.

    Remove Front Differential Case Front Outer Race Bearing

  1. Using SST 09387-00041 (09387-02020), remove the front differential case front outer race bearing from the front transaxle case.

    Install Shift Fork Shaft Bearing

  1. Using SST 09820-00031 and a press, press in the 4 shift fork shaft bearings.

    Install Output Shaft Cover

  1. Install the output shaft cover, as shown in the illustration.

    Install no.2 Output Shaft Front Bearing

  1. Using SST 09950-70010 (09951-07100, 09951-00710) and a hammer, install the No.2 output shaft front bearing.

    Install Output shaft cover

  1. Install the output shaft cover, as shown in the illustration.

    Install Output Shaft Front Bearing

  1. Coat a new output shaft bearing with gear oil, using SST09223-15020 and a press, install it.

    Install Bearing Lock Plate Bolt

  1. Install the bearing lock plate bolt. Torque 11 Nm.

    Install Transaxle Case Oil Seal

  1. Using SST 09950-60010 (09551-00580) and a hammer, install a new transaxle case oil seal to the front transaxle case.

    2. NOTE: Drive in depth: 4.4 to 5.0 mm (0.1732 to 0.1969 in.).

  2. Coat a new oil seal lip with MP grease.

    Install Input Shaft Front Bearing

  1. Coat a new input shaft front bearing with gear oil, using SST 09223-00010 and a press, install it.

    Install Bearing Lock Plate Bolt

  1. Install the bearing lock plate bolt. Torque 11 Nm.

    Install Input Shaft Rear Bearing Shaft Snap Ring

  1. Using snap ring pliers, install the 2 front bearing shift snap rings.

    Install Shift and Select Lever Shaft Needle Roller Bearing

  1. Using SST 09285-76010 and a press, press in a new shift and select lever shaft needle roller bearing.

    2. NOTE: Clearance: 177.8 to 178.7 mm (7.0000 to 7.0354 in.).

    Install Shift Fork Shaft Bearing

  1. Using SST 09307-12010 09820-00031 and a press, press in 4 new shift fork shaft bearings.

    2. NOTE: Clearance A: 168.2 to 169.3 mm (6.6220 to 6.6653 in.).

    NOTE: Clearance B: 162.2 to 163.3 mm (6.3858 to 6.4291 in.).

    Install Front Differential Case Front Tapered Roller Bearing

  1. Using SST 09550-70010 (09951-00910, 09951-07150) and hammer, install the front differential case front tapered roller bearing.

    Install Front Differential Case Rear Shim

  1. Install the front differential case rear shim into the manual transmission case.

    Install Front Differential Case Rear Tapered Roller Bearing

  1. Using SST 09950-60010 (09551-00910) 09950-70010 (09551-07100) and a hammer, install the front differential case rear tapered roller bearing.

    Adjust Differential Side Bearing Preload

  1. Install differential case assembly to the front transaxle case.
  2. Install the manual transmission case to the manual transaxle case with the 12 bolts. Torque 29 Nm.
  3. Install the 6 bolts of the manual transaxle case. Torque 29 Nm.
  4. Turn the differential case in both directions to get it to smooth.
  5. Using SST 09564-32011 and a torque wrench, measure the starting preload.

    6. NOTE: New bearing preload: 1.00 to 2.49 N*m (10.2 to 25.4 kgf*cm, 8.85 to 22.04 in.*lbf).

  6. Select a new shim.
  7. Remove the manual transmission case from the front transaxle case.
  8. Remove the 18 bolts and manual transmission case.

    Install no.2 Output Shaft Rear Bearing Outer Race

  1. Install the output shaft rear bearing shim onto the manual transmission case.

    2. NOTE: When reusing the output shaft rear tapered roller bearing, first install a shim of the same thickness as the original. When installing a new output shaft front bearing, first select and install a shim which is thinner than the original.

  2. Using SST 09950-60010 (09951-00650) 09950-70010 (09951-07200) and a press, press in the output shaft rear bearing outer race.

    Adjust no.2 Output Shaft Rear Bearing Preload

  1. Install the No.2 output shaft and the differential case onto the front transaxle case.
  2. Install the manual transmission case and 12 bolts to the front transaxle case. Torque 29 Nm.
  3. Install the 6 bolts of the manual transaxle case side. Torque 29 Nm.
  4. Measure the No.2 output shaft rear bearing preload. Subtract the value of the differential side bearing preload from the measured No.2 output shaft rear bearing preload. Select a shim that will set the preload within the specified range below and install the shim.

    5. NOTE: New bearing preload: 3.9 to 5.5 N*m (34 to 40 kgf*cm, 48 to 56 in.*lbf).

  5. Select a new shim.
  6. Remove the manual transmission case.
  7. Remove the 18 bolts and manual transmission case.
  8. Remove the output shaft assembly from the front transaxle case.
  9. Remove the differential case from the front transaxle case.

    Install Transmission Magnet

  1. Clean and reinstall the transmission magnet onto the front transaxle case.

    Install Front Differential Case Assembly

  1. Coat the differential case tapered roller bearing with gear oil, and install it onto the front transaxle case.

    Install Transmission Oil Separator

  1. Install 2 bolts and transmission oil separator onto the front transaxle case. Torque 8.5 Nm.

    Install Input Shaft, no.1 Output Shaft and no.2 Output Shaft

  1. Install the 3 shafts at the same time.

    Install Reverse Idler Gear

  1. Coat the reverse idler gear, needle roller bearing and reverse idler thrust washer with MP grease, install them to the reverse idler gear shaft.

    2. NOTE: Make sure that the protruding part on the reverse idler thrust washer is fitted into the groove of the reverse idler shaft.

  2. Install the reverse idler gear shaft by sliding and lifting it.
  3. Align the mark of the reverse idler gear shaft with the hole of the bolt.

    Install Reverse Shift Fork

  1. Install the No.1 gear shift fork onto the No.4 hub sleeve.

    Install no.3 Gear Shift Fork

  1. Install the No.1 gear shift fork onto the No.3 hub sleeve.

    Install 5th and 6th Shift Fork Shaft

  1. Install the 5th and 6th shift fork shaft onto the front transaxle case.
  2. Install the bolt into the No.3 gear shift fork. Torque 20 Nm.

    Install Reverse Shift Fork shaft

  1. Install the reverse shift fork shaft into the front transaxle case.
  2. Install the bolt into the No.4 gear shift fork. Torque 20 Nm.

    Install Shift Arm

  1. Install the shaft arm into the front transaxle case.

    Install no.1 Gear Shift Fork

  1. Install the No.1 gear shift fork onto the No.1 hub sleeve.

    Install no.5 Gear Shift Fork Shaft

  1. Install the No.3 gear shift head to the No.5 gear shift fork shaft.
  2. Using a pin punch and hammer, drive the slotted pin into the No.3 gear shift head.

    3. NOTE: Depth: -0.5 to 0.5 mm (-0.0197 to 0.0197 in.).

  3. Install the No.5 gear shift fork shaft into the front transaxle case.
  4. Assemble the shift arm and No.5 gear shift fork shaft.
  5. Install the shift arm pin into the shift arm.
  6. Using a brass bar and hammer, install a new E-ring onto the shift arm pin.

    Install no.2 Shift Fork

  1. Install the No.2 gear shift fork onto the No.2 hub sleeve.

    Install 3rd and 4th Shift Fork shaft

  1. Install the No.3 gear shift head to the 3rd and 4th shift fork shaft.
  2. Install the 3rd and 4th shift fork shaft to the transmission case.
  3. Using a pin punch and hammer, drive a new slotted spring pin into the No.2 gear shift fork.

    4. NOTE: Drive in depth : -0.5 to 0.5 mm (-0.0197 to 0.0197 in.).

  4. Using a pin punch and hammer, drive the slotted spring pin into the No.3 gear shift head.

    5. NOTE: Drive in depth : -0.5 to 0.5 mm (-0.0197 to 0.0197 in.).

    Install 1st and 2nd Shift Fork Shaft

  1. Install the 1st and 2nd shift fork shaft into the front transaxle case.
  2. Install the bolt onto the No.1 gear shift fork. Torque 20 Nm.

    Install Transmission Oil Separator

  1. Install the transmission oil separator onto the manual transmission case with the 2 bolts. Torque 8.5 Nm

    Install no.1 Oil Receiver Pipe

  1. Install the No.1 oil receiver pipe onto the manual transmission case.

    2. CAUTION: Do not damage the No.1 oil receiver pipe.

    Install Manual Transmission Case

  1. Apply FIPG to the manual transmission case, as shown in the illustration.

    2. CAUTION: Assemble parts within 10 minutes of application. Otherwise, the packing (FIPG) material must be removed and reapplied.

    NOTE: FIPG: Part No. 08826-00090, THREE BOND 1281 or equivalent.

  2. Use 2 snap ring pliers to keep the snap ring stretched and install the manual transmission case.
  3. Install a bolt to the No.1 output shaft and lift the No.1 output shaft from the service hole. Make sure that the snap rings are positioned correctly in the bearing grooves by checking that the distances between the centers of snap ring holes are as shown in the illustration.
  4. Install the manual transmission case and 12 bolts to the front transaxle case. Torque 29 Nm.
  5. Install the manual transmission case and 6 bolts to the front transaxle case. Torque 29 Nm.

    Install Reverse Idler Gear Shaft Bolt

  1. Coat the reverse idler gear shaft bolt with sealant, and install it with a new gasket. Torque 80 Nm.

    2. NOTE: Sealant: Part No. 08833-00080, THREE BOND 1344, LOCTITE 242 or equivalent.

    Install Ball

  1. Coat the 4 shift detent ball plugs with sealant.

    2. NOTE: Sealant: Part No. 08833-00080, THREE BOND 1344, LOCTITE 242 or equivalent.

  2. Install the 4 balls, 4 spring seats and 4 shift detent ball plugs with using a hexagon wrench (6 mm). Torque 22 Nm.

    Install Manual Transmission Case Plug

  1. Apply FIPG to the manual transmission case, as shown in the illustration.

    2. NOTE: FIPG: Part No. 08826-00090, THREE BOND 1281 or equivalent.

  2. Coat the transmission case plug with sealant.

    3. NOTE: Sealant: Part No. 08833-00080, THREE BOND 1344, LOCTITE 242 or equivalent.

  3. Install the manual transmission case plug into the manual transmission case. Case plug 1.
    Torque 55 Nm. Case plug 2.
    Torque 22 Nm.

    Install O-Ring

  1. Coat the O-ring with the grease, and install it into the shift and select pin.

    Install Shift and Select Pin

  1. Install the spring.
  2. While pressing the shift and select pin (*1) in as far as it will go, push in the shift and select pin (*2).

    Install Check Valve

  1. Install the check valve with a new gasket.

    Install Temperature Sensor

  1. Install the temperature sensor with a new gasket.

    Install Transmission Case Oil Seal

  1. Coat a new oil seal lip with MP grease.
  2. Using SST and a hammer, install the transmission case oil seal.

    3. CAUTION: Do not damage the oil seal lip.

    NOTE: Drive in depth: -0.5 to 0.5 mm (-0.0197 to 0.0197 in.).

    Install Transaxle Case Oil Seal

  1. Coat a new oil seal lip with MP grease.
  2. Using SST and a hammer, install the transaxle case oil seal.

    3. CAUTION: Do not damage the oil seal lip.

    NOTE: Drive in depth: -0.5 to 0.5 mm (-0.0197 to 0.0197 in.).

    Install Control Shaft Cover

  1. Align the 4 shift fork shafts as shown in the illustration.
  2. Apply FIPG to the manual transmission case, as shown in the illustration.

    3. NOTE: FIPG: Part No. 08826-00090, THREE BOND 1281 or equivalent.

  3. Coat the 4 bolts with sealant.

    4. NOTE: Sealant: Part No. 08833-00080, THREE BOND 1344, LOCTITE 242 or equivalent.

  4. Install the shift and select lever shaft onto the manual transmission case with the 4 bolts. Torque 19 Nm.

    Install Shift Gate Pin

  1. Coat the shift gate pin with sealant.

    2. NOTE: Sealant: Part No. 08833-00080, THREE BOND 1344, LOCTITE 242 or equivalent.

  2. Install the shift gate pin onto the manual transmission case. Torque 30 Nm.

    Install no.2 Lock Ball Assembly

  1. Coat the No.2 lock ball assembly with sealant.

    2. NOTE: Sealant: Part No. 08833-00080, THREE BOND 1344, LOCTITE 242 or equivalent.

  2. Install the No.2 lock ball assembly onto the manual transmission case. Torque 29 Nm.

    Install no.1 Lock Ball Assembly

  1. Coat the No.1 lock ball assembly with sealant.

    2. NOTE: Sealant: Part No. 08833-00080, THREE BOND 1344, LOCTITE 242 or equivalent.

  2. Install the No.1 lock ball assembly onto the manual transmission case. Torque 39 Nm.

    Install Back-up Light Switch Assembly

  1. Install the back-up light switch assembly onto the manual transmission case using SST 09816-30010 . Torque 40 Nm.

    Install Selecting Bell Crank Assembly

  1. Coat the 2 bolts with sealant, install it with selecting bell crank assembly to the manual transmission case. Torque 20 Nm.

    2. NOTE: Sealant: Part No. 08833-00080, THREE BOND 1344, LOCTITE 242 or equivalent.

    Install Floor Shift Control Lever Housing Support Bracket

  1. Install the floor shift control lever housing support bracket onto the manual transaxle case with the 3 bolts. Torque 17 Nm.

    2. NOTE: Sealant: Part No. 08833-00080, THREE BOND 1344, LOCTITE 242 or equivalent.

    Install Clutch Release Fork Boot

  1. Install the clutch release fork boot onto the manual transmission case.

    Install Release Fork Support

  1. Using a deep socket wrench, install the release fork support to the front transaxle case. Torque 37 Nm.

    Install Clutch Release Bearing Assembly

  1. Apply the release hub grease to the release fork and release bearing assembly contact, release fork and push rod contact, and release fork pivot points.

    2. NOTE: Sealant: Part No. 08887-01806, RELEASE HUB GREASE or equivalent.

  2. Apply clutch sealant grease to the spline coupling part.

    3. NOTE: Sealant: Part No. 08887-01706, CLUTCH SPLINE GREASE or equivalent.

    Install Clutch Release Fork Sub-Assembly

  1. Install the bearing to the release fork, and then install them to the transaxle assembly.

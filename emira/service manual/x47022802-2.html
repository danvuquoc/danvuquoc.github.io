Cover - Dual Clutch - 4 Cylinder

Print

Operation Code: 47.02.28-02

Important Information

WARNING: Transmission oil is a toxic substance and can be lethal if ingested. Wear protective gloves, protective clothing and safety glasses.

    Removal

  1. Remove transmission assembly. Refer to procedure.
  2. Install transmission assembly to a suitable stand.
  3. Remove retaining ring securing dual clutch cover to transmission.

    4. NOTE: Do not damage seat of retaining ring and sealing surface on housing otherwise leaks may occur.

  4. Using a suitable tool, remove dual clutch cover.

    Installation

  1. Installation is the reverse of removal procedure except for the following:

    2. NOTE: Inspect retaining ring for damage, renew if necessary.

    NOTE: Ensure retaining ring is seated correctly.

  2. Clean sealing surface on housing.

    3. WARNING: Only use cleaning agents and solvents in a well ventilated area.

  3. Install assembly sleeve onto drive shaft to protect sealing lip on cover.
  4. Install dual clutch cover.
  5. Remove assembly sleeve from drive shaft.

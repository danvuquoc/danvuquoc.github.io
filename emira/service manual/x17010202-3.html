Harness - Engine - V6

Print

Operation Code: 17.01.02-02

    Removal

  1. Disconnect battery. Refer to procedure.
  2. Remove supercharger. Refer to procedure.
  3. Remove rear wheelarch liner - left side. Refer to procedure.
  4. Remove air conditioning compressor. Refer to procedure.
  5. Remove bootbox access panel. Refer to procedure.
  6. Disconnect harness connector from engine control module.
  7. Disconnect harness connector from engine bay harness.
  8. Disconnect harness connector from expansion tank.
  9. Disconnect harness connector from mass airflow sensor.
  10. Release clips (x4) securing harness to airbox.
  11. Disconnect harness connector from transmission.
  12. Release clips (x2) securing harness to transmission.
  13. Release clips (x2) securing harness to engine bay.
  14. Disconnect harness connector from starter motor.
  15. Disconnect harness connector from fuel sensor.
  16. Disconnect harness connector from coolant temperature sensor.
  17. Release clips (x2) securing harness to hose.
  18. Remove M6x12 bolt securing harness to engine.
  19. Release clips (x3) securing harness to engine.
  20. Disconnect harness connectors (x2) from engine.
  21. Release clips (x3) securing harness to engine.
  22. Disconnect harness connector from differential pressure sensor.
  23. Release clip securing harness to engine.
  24. Disconnect harness connector from camshaft position sensor.
  25. Disconnect harness connector from oil control valve.
  26. Release clips (x2) securing harness to engine.
  27. Disconnect harness connector from oil level sensor.
  28. Release clip securing harness to engine.
  29. Disconnect harness connector from crankshaft position sensor.
  30. Disconnect harness connector from alternator.
  31. Release clip securing harness to alternator.
  32. Disconnect harness connectors from camshaft position sensor.
  33. Disconnect harness connector from condenser.
  34. Release clips (x2) securing harness to engine.
  35. Disconnect harness connectors (x3) from ignition coils.
  36. Disconnect harness connector from oil control valve.
  37. Disconnect harness connector from camshaft position sensor.
  38. Remove M6x12 bolt securing harness to engine.
  39. Release clips (x3) securing harness to engine.
  40. Disconnect harness connectors (x6) from fuel injectors.
  41. Release clips (x6) securing harness to fuel rail.
  42. Release clip securing harness to engine.
  43. Disconnect harness connectors (x3) from ignition coils.
  44. Disconnect harness connector from camshaft position sensor.
  45. Remove engine harness.

    Installation

  1. Installation is the reverse of removal procedure.

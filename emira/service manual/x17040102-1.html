Switch - Gear Select Controls

Print

Operation Code: 17.04.01-02

    Removal

  1. Remove rubber pad from centre console.
  2. Disengage clips (x11) securing centre console finisher to centre console.
  3. Remove centre console finisher.
  4. Remove 5x25 self tapping screws (x4) securing gear selector to centre console mounting.
  5. Remove gear selector to access connector.
  6. Disengage harness connector from gear selector.

    Installation

  1. Installation is the reverse of removal procedure except for the following:
  2. After installation perform a diagnostic read and clear error memory using Lotus Insight tool.

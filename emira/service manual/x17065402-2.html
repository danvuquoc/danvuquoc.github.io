Temperature Sensor - Catalytic Converter - 4 Cylinder

Print

Operation Code: 17.06.54-02

    Removal

  1. Remove rear undertray. Refer to procedure.
  2. Remove engine cover. Refer to procedure.
  3. Release union securing temperature sensor to catalytic converter.
  4. Release union securing temperature sensor to catalytic converter. Torque 45 Nm.
  5. Release clips (x2) securing temperature sensor to engine.
  6. Disconnect harness connector from temperature sensor.
  7. Release clip securing harness connector to engine.
  8. Remove temperature sensor.

    Installation

  1. Installation is the reverse of removal procedure except for the following:
  2. After installation perform a diagnostic read and clear error memory using Lotus Insight tool.

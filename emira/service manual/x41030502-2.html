Pulley - Crankshaft / Vibration Damper - 4 Cylinder

Print

Operation Code: 41.03.05-02

    Removal

  1. Remove drive belt. Refer to procedure.
  2. Install counterholder to crankshaft pulley / vibration damper.
  3. Remove and discard bolts (x4) securing crankshaft pulley / vibration damper to crankshaft. Torque 20 Nm + 90 degrees.

    4. WARNING: These fasteners have been torqued to a specification where the threads may have been damaged or stretched. Always discard and replace with new on installation.

    NOTE: Vehicle not shown for clarity.

  4. Remove crankshaft pulley / vibration damper.

    Installation

  1. Installation is the reverse of removal procedure except for the following:
  2. Installation position is specified by a pilot bushing.
  3. Replace discarded bolts.

Wire Assembly - V6

Print

Operation Code: 47.02.05-02

    Removal

  1. Remove solenoid assembly. Refer to procedure.
  2. Disconnect the 9 connectors.
  3. Remove the transmission wire from the clamp.
  4. Remove the bolt, lock plate and transmission wire from the valve body.

    Installation

  1. Coat the O-ring with ATF.
  2. Coat the bolt with ATF.
  3. Install the transmission wire and lock plate to the valve body with the bolt.
  4. Install the transmission wire to the clamp.
  5. Connect the 9 connectors.
  6. Perform a diagnostic read and clear memory using Lotus Insight tool.

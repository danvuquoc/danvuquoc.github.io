Wheel Speed Sensor - Front - Left Side - V6

Print

Operation Code: 17.06.34-02

    Remove

  1. Remove road wheel front - left side. Refer to procedure.
  2. Disconnect wiring harness from harness mounting clip and wheel speed sensor.
  3. Move wiring harness aside.
  4. Remove M6x16 bolt securing wheel speed sensor to hub carrier. Torque 8 Nm.
  5. Remove wheel speed sensor from hub carrier.

    Installation

  1. Installation is the reverse of removal procedure except for the following:
  2. Perform a diagnostic read and clear error memory using Lotus Insight tool.

Electronic Control Module (ECM) - V6

Print

Operation Code: 17.06.13-02

    Removal

  1. Disconnect battery. Refer to procedure.
  2. Remove rear wheelarch liner left side. Refer to procedure.
  3. Remove transmission control unit. Refer to procedure.

    4. NOTE: Only applies to vehicles fitted with automatic transmission.

  4. Disconnect harness connectors (x3) secured to ECM and move harness aside.
  5. Remove M6 nuts (x3) securing ECM to bracket. Torque 4.5 Nm.
  6. Remove ECM.

    Installation

  1. Installation is the reverse of removal procedure.
  2. After installation download software using Emira DS.
  3. Perform a diagnostic read and clear error memory using Lotus Insight tool.

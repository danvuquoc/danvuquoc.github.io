Condenser Assembly

Print

Operation Code: 18.03.07-02

    Removal

  1. Remove radiator assembly. Refer to procedure.
  2. Remove bolts (x6) securing condenser and power steering fluid return pipe to radiator.
  3. Remove condenser and power steering fluid return pipe assembly.
  4. Remove bolt securing bracket to condenser.
  5. Remove bracket.
  6. Disengage fir tree clips (x2) securing power steering fluid return pipe to condenser.
  7. Remove bolt securing power steering fluid return pipe to condenser.
  8. Remove power steering return pipe.

    Installation

  1. Installation is the reverse of removal procedure.

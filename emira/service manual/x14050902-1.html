Finisher - Fascia End - Left Side

Print

Operation Code: 14.05.09-02

    Removal

  1. Partially pull back door seal to access finisher.
  2. Release finisher from lower edge to disengage clips (x2).
  3. Refer to illustration for clip type and location.

    4. CAUTION: Top edge of finisher hooks under fascia top panel.

  4. Remove finisher from vehicle.

    Installation

  1. Installation is the reverse of removal procedure.

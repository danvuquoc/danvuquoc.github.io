Switch - Engine Start

Print

Operation Code: 17.05.20-02

    Removal

  1. Disconnect battery. Refer to procedure.
  2. Remove rubber pad from centre console.
  3. Release clips (x11) securing centre console finisher to centre console.

    4. NOTE: Release clips working from front to rear.

  4. Remove centre console finisher.
  5. Disengage clips (x2) securing engine start switch to centre console and remove engine start switch from centre console.
  6. Disconnect harness connector from engine start switch.
  7. Remove engine start switch.

    Installation

  1. Installation is the reverse of removal procedure.

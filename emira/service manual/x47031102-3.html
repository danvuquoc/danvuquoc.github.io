Transmission Oil Level Check - Manual - V6

Print

Operation Code: 47.03.11-02

    Check

  1. Remove rear engine bay trim cover. Refer to procedure.
  2. Fabricate dipstick to specifications.

    3. NOTE: 60mm dipstick length with extra material to have finger grip and a stop plate.

  3. Remove M6x16 bolt securing transmission oil return plug to transmission.
  4. Remove transmission oil return plug from transmission.
  5. Insert dipstick.
  6. Remove dipstick.

    7. NOTE: If there is oil on the dipstick, there is sufficient oil.

  7. Install transmission oil return plug to transmission.
  8. Install M6x16 bolt securing transmission oil return plug to transmission. Torque 7 Nm.

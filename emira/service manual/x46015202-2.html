Hose - Auxiliary Radiator Outlet - 4 Cylinder

Print

Operation Code: 46.01.52-02

    Removal

  1. Drain coolant. Refer to procedure.
  2. Release clamp securing auxiliary radiator outlet hose to auxiliary radiator intermediate outlet pipe.
  3. Disconnect auxiliary radiator outlet hose from auxiliary radiator intermediate outlet pipe.

    4. WARNING: Coolant is a toxic substance and can be lethal if ingested.

    NOTE: Drain coolant into a suitable container and dispose of appropriately.

  4. Release clamp securing auxiliary radiator outlet hose to auxiliary radiator.
  5. Disconnect auxiliary radiator outlet hose from auxiliary radiator.
  6. Remove auxiliary radiator outlet hose.

    Installation

  1. Installation is the reverse of removal procedure except for the following:
  2. Refill coolant.

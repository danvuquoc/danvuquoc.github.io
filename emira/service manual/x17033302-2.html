Module - Central Powertrain Controller (CPC) - 4 Cylinder

Print

Operation Code: 17.03.33-02

    Removal

  1. Disconnect battery. Refer to procedure.
  2. Remove rear quarter trim. Refer to procedure.
  3. Disconnect harness connectors (x3) from Central Powertrain Controller (CPC) module.
  4. Remove M5x16 bolts (x2) securing CPC module to bracket.
  5. Release CPC module from bracket.
  6. Remove CPC module.

    Installation

  1. Installation is the reverse of removal procedure.
  2. Perform a diagnostic read and clear error memory using Lotus Insight tool.

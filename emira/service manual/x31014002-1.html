Road Spring - Front - Left Side

Print

Operation Code: 31.01.40-02

    Remove

  1. Remove spring / damper assembly - left side. Refer to procedure.
  2. Install spring compressor to road spring.

    3. NOTE: Make sure spring tension is fully restrained before dismantling any further.

  3. Remove M12 nut and M12x90 bolt securing damper to mounting bracket. Torque 120 Nm.
  4. Remove mounting bracket and upper spring seat from damper.
  5. Remove road spring from damper.

    Installation

  1. Installation is the reverse of the removal procedure.

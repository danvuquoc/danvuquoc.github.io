Body G-Satellite Sensor - Left Side

Print

Operation Code: 17.35.13-02

    Removal

  1. Remove rear quarter trim panel. Refer to procedure.
  2. Disconnect harness connector from GSAT sensor.
  3. Remove M6x25 bolt securing body GSAT sensor to bracket. Torque 10 Nm.
  4. Remove body GSAT sensor.

    Installation

  1. Installation is the reverse of removal procedure except for the following:
  2. Perform a diagnostic read and clear error memory using Lotus Insight tool.
  3. Turn on ignition and make sure SRS warning lamp extinguishes.

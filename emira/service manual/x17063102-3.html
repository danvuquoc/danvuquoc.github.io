Sensor - Boost Temperature Pressure - V6

Print

Operation Code: 17.06.31-02

    Removal

  1. Disconnect battery. Refer to procedure.
  2. Remove front engine bay cover. Refer to procedure.
  3. Disconnect boost sensor connector from supercharger.
  4. Remove M6x16 bolt securing boost sensor to supercharger. Torque 10 Nm.
  5. Remove boost sensor.

  2. After installation perform a diagnostic read and clear error memory using Lotus Insight tool.

Speed Sensor Assembly - Transmission - V6

Print

Operation Code: 47.02.06-02

    Removal

  1. Remove solenoid assembly. Refer to procedure.
  2. Disconnect the connector.
  3. Remove the 2 bolts and speed sensor from the valve body.

    Installation

  1. Coat the 2 bolts with ATF.
  2. Install the speed sensor to the valve body with the 2 bolts. Torque 11 Nm.
  3. Connect the connector.
  4. Perform a diagnostic read and clear memory using Lotus Insight tool.

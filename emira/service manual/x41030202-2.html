Auxiliary Drive Belt - 4 Cylinder

Print

Operation Code: 41.03.02-02

    Removal

  1. Remove rear undertray. Refer to procedure.
  2. Remove rear wheel arch liner - right side.

    3. NOTE: Procedure is the same as left side component.

  3. Rotate belt tensioner clockwise to relieve tension from drive belt.

    4. NOTE: If reusing belt, take note of and reinstall in the original direction

    NOTE: Vehicle not shown for clarity.

  4. Use allen key to lock tensioner in fully released position.
  5. Remove belt from alternator pulley.
  6. Remove belt from AC compressor pulley.
  7. Remove belt from crankshaft pulley.
  8. Remove auxiliary drive belt.

    Installation

  1. Install belt to crankshaft pulley.
  2. Install belt to AC compressor pulley.
  3. Install belt to alternator pulley.
  4. Remove allen key.
  5. Rotate belt tensioner counter clockwise.
  6. Check belt position.
  7. Start engine and observe belt and pulley operation and direction.

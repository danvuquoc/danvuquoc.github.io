Crankshaft Position Sensor - 4 Cylinder

Print

Operation Code: 17.06.03-02

    Removal

  1. Remove rear undertray. Refer to procedure.
  2. Disconnect harness connector from crankshaft position sensor.
  3. Remove M6 bolt securing crankshaft position sensor to engine. Torque 8 Nm.
  4. Remove crankshaft position sensor.

    Installation

  1. Installation is the reverse of removal procedure except for the following:
  2. Apply engine oil to O-ring before installation.
  3. After installation perform a diagnostic read and clear error memory using Lotus Insight tool.

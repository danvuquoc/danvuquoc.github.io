Road Wheel - Rear - Left Side

Print

Operation Code: 34.01.11-01

    Removal

  1. Raise and support vehicle on 2 post lift. Refer to procedure.
  2. Remove wheel bolts (x5) securing road wheel to hub assembly. Torque 105 Nm.

    3. CAUTION: Use the correct tool to remove locking wheel nuts if installed.

  3. Remove road wheel from hub assembly.

    Installation

  1. Install wheel to hub assembly and align wheel bolt holes with hub.

    2. NOTE: Make sure tyre rotation direction is correct when you install the wheel.

  2. Install wheel bolts (x5) and tighten lightly.
  3. Lower the vehicle to the ground and torque wheel bolts in a diagonal sequence.

    4. NOTE: Make sure the locking wheel nut tool is installed in the vehicle after use.

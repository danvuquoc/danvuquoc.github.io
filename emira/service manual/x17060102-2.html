Engine Coolant Temperature Sensor - 4 Cylinder

Print

Operation Code: 17.06.01-02

    Removal

  1. Allow engine to cool completely.
  2. Remove rear bulkhead access panel. Refer to procedure.
  3. Disconnect harness connector from engine coolant temperature sensor.
  4. Remove M6 bolt securing engine coolant temperature sensor to engine. Torque 9 Nm.
  5. Remove engine coolant temperature sensor.

    6. NOTE: Be prepared to catch any excess fluid.

    Installation

  1. Installation is the reverse of removal procedure except for the following:
  2. After installation perform a diagnostic read and clear error memory using Lotus Insight tool.

Glove Box Lid

Print

Operation Code: 14.12.02-02

    Removal

  1. Remove glove box assembly. Refer to procedure.
  2. Disengage glove box lid from guide rails (x2).
  3. Remove retaining pins (x2) securing glove box lid to glove box liner.

    4. NOTE: Retain retaining pins for installation purposes.

  4. Remove glove box lid.

    Installation

  1. Installation is the reverse of removal procedure.

Temperature Sensor - Coolant Thermostat - 4 Cylinder

Print

Operation Code: 17.06.58-02

    Removal

  1. Allow engine to cool completely.
  2. Remove charge air distributor. Refer to procedure.
  3. Disconnect harness connector from temperature sensor.
  4. Release clip securing temperature sensor to thermostat.
  5. Remove temperature sensor.

    6. NOTE: Be prepared to catch any excess fluid.

  6. Remove and discard O-ring.

    Installation

  1. Installation is the reverse of removal procedure except for the following:
  2. Renew discarded O-ring.
  3. After installation perform a diagnostic read and clear error memory using Lotus Insight tool.

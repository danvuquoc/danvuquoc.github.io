Module - Electronic Steering Lock

Print

Operation Code: 32.05.23-02

    Removal

  1. Remove upper steering column assembly. Refer to procedure.
  2. Remove M5x16 Torx screws (x3) securing electronic steering lock module to upper steering column. Torque 5 Nm.

    3. Note: Tighten bolts in numerical order.

  3. Remove electronic steering lock module.
  4. Release harness connector from electronic steering lock module.
  5. Remove harness adapter.

    Installation

  1. Installation is the reverse of removal procedure except for the following:
  2. After installation perform a diagnostic read and clear error memory using Lotus Insight tool.

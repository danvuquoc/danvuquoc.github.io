Oxygen Sensor - Pre Catalytic Converter - 4 Cylinder

Print

Operation Code: 17.06.05-02

    Removal

  1. Remove engine cover. Refer to procedure.
  2. Disconnect harness connector from oxygen sensor.
  3. Release oxygen sensor connector from bracket.
  4. Release clips (x2) securing oxygen sensor to engine mount.
  5. Unscrew oxygen sensor from catalytic converter. Torque 45 Nm.
  6. Remove oxygen sensor.

  1. Installation is the reverse of removal procedure except for the following:
  2. After installation perform a diagnostic read and clear error memory using Lotus Insight tool.

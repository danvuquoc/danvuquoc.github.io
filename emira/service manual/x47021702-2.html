Control Unit - 4 Cylinder

Print

Operation Code: 47.02.17-02

Important

WARNING: Transmission oil is a toxic substance and can be lethal if ingested. Wear protective gloves, protective clothing and safety glasses.

    Removal

  1. If control unit is to be renewed, read out control unit and save adaptation values.
  2. Remove controller unit assembly. Refer to procedure.
  3. Remove and discard M4 bolts (2) (x14) securing control unit (N15/15n1) to controller unit assembly (3). Torque 2 Nm + 90 degrees.

    4. CAUTION: These fasteners have been torqued to a specification where the threads may have been damaged or stretched. Always discard and replace with new on installation.

  4. Remove bolts (4) (x3) securing tandem oil pump to control unit. Torque 5 Nm.
  5. Remove control unit.
  6. Remove and discard O-rings (5) (x2).

    Installation

  1. Installation is the reverse of removal procedure except for the following:

    2. NOTE: Ensure control unit is positioned correctly on controller unit assembly.

  2. Renew discarded O-rings.
  3. Apply transmission oil to O-rings.
  4. Renew discarded bolts.
  5. Transfer stored adaptation values to control unit and perform SCN coding using Lotus Insight tool.
  6. Perform a diagnostic read and clear error memory using Lotus Insight tool.

Glove Box Inner Liner

Print

Operation Code: 14.12.03-02

    Removal

  1. Remove glove box assembly. Refer to procedure.
  2. Disengage glove box from guide rails (x2).
  3. Remove retaining pins (x2) securing glove box lid to glove box liner.

    4. NOTE: Retain retaining pins for installation purposes.

  4. Remove glove box inner liner.

    5. Do not carry out further disassembly if removing for access only.

  5. Disengage trim clips and cable ties (x7) securing glove box harness to glove box inner liner.
  6. Disconnect harness connectors (x5) secured to glove box inner liner.
  7. Remove glove box harness.
  8. Disengage clips (x4) and remove footwell light from glove box inner liner.
  9. Disengage clips (x2) and remove airbag defeat switch from glove box inner liner.
  10. Remove self tapping screws (x2) and remove glove box damper from glove box inner liner.
  11. Remove self tapping screws (x3) and remove latch assembly from glove box inner liner.
  12. Disengage clips (x4) and remove interior light from glove box inner lining.
  13. Remove self tapping screws (x2) and remove stopper bracket and sliding door switch from glove box inner lining.

    Installation

  1. Installation is the reverse of removal procedure except for the following:
  2. Renew all discarded cable ties.

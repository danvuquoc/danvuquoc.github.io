Door Latch Module - Left Side

Print

Operation Code: 10.17.19-02

    Removal

  1. Partially remove door water deflector panel left side. Refer to procedure.
  2. Disengage clip securing door latch module pull rod to lock assembly.

    3. NOTE: Only applicable for the left side door.

  3. Remove pull rod from door latch module.
  4. Disengage exterior door handle release cable from bracket.
  5. Disengage exterior door handle release cable from door latch module lever.
  6. Disconnect harness connector from door latch module.
  7. Remove M6x12 torx bolts (x3) securing door latch module to door panel. Torque 7 Nm.
  8. Remove door latch module.

    Removal

  1. Installation is the reverse of removal procedure.

Dual Clutch - 4 Cylinder

Print

Operation Code: 47.02.20-02

Important Information

WARNING: Transmission oil is a toxic substance and can be lethal if ingested. Wear protective gloves, protective clothing and safety glasses.

    Removal

  1. Remove dual clutch axial bearing. Refer to procedure.
  2. Pull locking pin (A) upwards.
  3. Fully open lever (B).
  4. Install assembly tool (01) onto dual clutch (5).
  5. Pull locking pin upwards again.
  6. Close lever and latch locking pin into assembly tool.
  7. Remove dual clutch (5) from clutch housing (27K) using assembly tool.

    8. NOTE: Discard sealing rings (6).

    Installation

  1. Installation is the reverse of removal procedure except for the following:

    2. NOTE: Renew sealing rings.

    NOTE: Ensure sealing rings are correctly seated.

  2. Install dual clutch into clutch housing carefully and without tilting using assembly tool.
  3. Rotate dual clutch (5) slightly until toothing of drive shaft (30) and hollow shaft (40) engages correctly in the dual clutch.
  4. Remove assembly tool from dual clutch.
  5. Measure installed dimension of dual clutch and adjust if necessary.

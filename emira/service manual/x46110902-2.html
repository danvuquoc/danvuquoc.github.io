Pump - 4 Cylinder

Print

Operation Code: 46.11.09-02

    Removal

  1. Remove rear bulkhead access panel. Refer to procedure.
  2. Remove rear undertray. Refer to procedure.
  3. Disconnect harness connector from coolant pump.
  4. Release clamps (x2) securing coolant hoses to coolant pump.
  5. Disconnect coolant hoses (x2) from coolant pump.
  6. Allow coolant to drain.

    7. WARNING: Coolant is a toxic substance and can be lethal if ingested.

    NOTE: Drain coolant into a suitable container and dispose of appropriately.

  7. Remove M6x12 bolts (x2) securing coolant pump to bracket. Torque 5 Nm.
  8. Remove upper bracket.
  9. Remove coolant pump.

    Installation

  1. Installation is the reverse of removal procedure except for the following:
  2. Refill coolant.
  3. Start engine and allow to idle.

    4. NOTE: Run the engine with the reservoir cap off to bleed the system.

  4. Refit coolant reservoir cap.

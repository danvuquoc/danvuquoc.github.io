Switch Assembly - Neutral Start - V6

Print

Operation Code: 47.02.02-02

    Removal

  1. Remove airbox assembly. Refer to procedure.
  2. Disconnect link rod from ball mounting on control shaft lever.
  3. Disconnect the connector from the park/neutral position switch.
  4. Remove the nut, washer and control shaft lever from the control shaft.
  5. Remove the 2 bolts and park/neutral position switch from the control shaft.

    6. CAUTION: Before removing the park/neutral position switch, remove any dirt or rust on the installation portion of the control shaft. Be sure to remove the switch straight along the shaft while being careful not to deform the plate spring that supports the shaft. If the plate spring is deformed, the park/neutral switch cannot be reinstalled correctly.

    Installation

  1. Move the shift lever to the N position.
  2. Align the protrusions of the park/neutral position switch.
  3. Install the park/neutral position switch to the control shaft with the 2 bolts. Torque 5.4 Nm.
  4. Install the control shaft lever to the control shaft with the nut and washer. Torque 13 Nm.
  5. Connect the connector to the park/neutral position switch.
  6. Connect the link rod to ball mounting on control shaft lever.

Clutch Assembly - AC Compressor - V6

Print

Operation Code: 40.23.03-02

    Removal

  1. Remove AC compressor. Refer to procedure.

    Installation

  1. Install circlip to stator.
  2. Install circlip to pulley.
  3. Install clutch shims.

    4. NOTE: Use at least the same amount of clutch shims as removed.

  4. Check clutch clearance.

    5. NOTE: Add clutch shims as required if there is no clearance.

  5. Connect AC compressor to suitable 12V power source.
  6. Use Dial Test Indicator (DTI) to measure distance between hub and rotor while operating AC clutch.

    7. NOTE: Distance should be between 0.26mm and 0.6mm between the clutch hub and the pulley.

    NOTE: Add shims if outside of clearance.

  7. If AC compressor clutch fails to operate, check resistance at following terminals:

    8. NOTE: If resistance is not as specified, on connectors replace the AC compressor and clutch assembly.

    NOTE: If resistance is not as specified at ground connector, replace AC clutch.

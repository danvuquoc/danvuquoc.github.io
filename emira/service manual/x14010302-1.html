Switch - Brake Light

Print

Operation Code: 14.01.03-02

    Removal

  1. Remove lower fascia panel. Refer to procedure.
  2. Disconnect battery. Refer to procedure.
  3. Disconnect harness connector from brake light switch.
  4. Disengage brake light switch from fixation ring.

    5. NOTE: Turn brake light switch counter clockwise to disengage.

  5. Remove brake light switch.

  2. After installation perform a diagnostic read and clear error memory using Lotus Insight tool.

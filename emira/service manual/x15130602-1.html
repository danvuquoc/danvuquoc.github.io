Interior Door Pull Trim - Left Side

Print

Operation Code: 15.13.06-02

    Removal

  1. Remove door trim assembly - left side. Refer to procedure.
  2. Remove M4x16 screws (x16) securing interior door pull trim to door trim.
  3. Pull electrical harness through door trim as you remove the interior door pull trim.
  4. Remove interior door pull trim from door trim.

    5. Do not carry out further disassembly if component is removed for access only.

  5. Release clips (x5) securing door harness to interior door pull trim.
  6. Disconnect harness connectors (x4) from switch pack and seat memory switch.
  7. Remove door harness.
  8. Remove M4x13 screws (x2) securing door pocket cover to interior door pull trim.
  9. Release clips (x9) securing door pocket cover to interior door pull trim.
  10. Remove door pocket cover.
  11. Release clips (x11) securing door pocket light guide to interior door pull trim.
  12. Remove door pocket light guide.
  13. Remove M4x13 screws (x4) securing switch panel trim to interior door pull trim.
  14. Remove switch panel trim.
  15. Remove M4x13 screws (x8) securing speaker grille to interior door pull trim.
  16. Release clips (x2) securing speaker grille to interior door pull trim.
  17. Remove speaker grille.

    Installation

  1. Installation is the reverse of removal procedure.

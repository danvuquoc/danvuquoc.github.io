Switch - Seat Memory

Print

Operation Code: 17.05.43-02

    Removal

  1. Remove interior door pull trim - left side. Refer to procedure.
  2. Disconnect harness connector from seat memory switch.
  3. Release clips (x4) securing seat memory switch to interior door pull trim.
  4. Remove seat memory switch.

    Installation

  1. Installation is the reverse of removal procedure.

Electronic Control Module (ECM) - 4 Cylinder

Print

Operation Code: 17.06.13-02

    Removal

  1. Disconnect battery. Refer to procedure.
  2. Remove engine cover. Refer to procedure.
  3. Remove splash shield secured to Electronic Control Module (ECM).

    4. NOTE: The splash shield is retained using adhesive strips.

  4. Disconnect harness connectors (x6) from ECM.
  5. Remove 6x20 screws (x4) securing ECM to airbox assembly. Torque 6 Nm.
  6. Remove ECM.

    Installation

  1. Installation is the reverse of removal procedure.
  2. Clean off excess adhesive from splash shield and ECM, renew double sided foam adhesive tape.
  3. After installation download software using Emira DS.
  4. Perform a diagnostic read and clear error memory using Lotus Insight tool.

Motor - Window Regulator - Left Side

Print

Operation Code: 11.02.09-02

    Removal

  1. Remove door trim assembly - left side. Refer to procedure.
  2. Disconnect harness connector from window motor.
  3. Remove 5x22 screws (x3) securing window motor to water deflector panel.
  4. Remove window motor from water deflector panel.

    5. Do not carry out further disassembly if component is removed for access only.

  5. Disengage clips (x2) securing module to window motor.
  6. Carefully pull module down to disengage module from window motor.
  7. Remove module from window motor.

    Installation

  1. Installation is the reverse of removal procedure except for the following:
  2. Check door glass alignment and reset window one touch facility. Refer to procedure.

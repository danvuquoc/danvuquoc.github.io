Filter and Check Valve - V6

Print

Operation Code: 46.09.04-02

    Removal

  1. Remove rear undertray. Refer to procedure.

    2. WARNING: Allow engine to cool completely before proceeding.

  2. Loosen nut securing reductor pipe to hose assembly. Torque 20 Nm.

    3. NOTE: Use a suitable container to catch any transmission oil.

  3. Disconnect hose assembly from reductor pipe.
  4. Remove filter/check valve from reductor pipe.

    5. NOTE: Use a suitable container to catch any transmission oil.

    Installation

  1. Installation is the reverse of removal procedure except for the following:
  2. Renew filter or clean filter with suitable degreaser.
  3. Check transmission oil level. Refer to procedure.

Pressure and Temperature Sensor - 4 Cylinder

Print

Operation Code: 17.06.51-02

    Removal

  1. Allow engine to cool completely.
  2. Remove charge air distributor. Refer to procedure.
  3. Disconnect harness connector from pressure and temperature sensor.
  4. Use a suitable tool to remove pressure and temperature sensor. Torque 15 Nm.

    5. NOTE: Be prepared to catch any excess fluid.

    Installation

  1. Installation is the reverse of removal procedure except for the following:
  2. After installation perform a diagnostic read and clear error memory using Lotus Insight tool.

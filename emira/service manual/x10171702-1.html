Door Release Handle - Interior - Left Side

Print

Operation Code: 10.17.17-02

    Removal

  1. Remove door trim assembly - left side. Refer to procedure.
  2. Remove 3x8 self tapping screw securing interior trim finisher to interior door release handle.

    3. NOTE: Pull insulation trim aside to access screws.

  3. Release trim clips (x4) and carefully pull trim finisher away from interior door release handle.
  4. Disconnect harness connector from lamp.
  5. Remove 4x13 self tapping screws (x4) securing interior door release handle to door trim.
  6. Remove interior door release handle from door trim.

    7. Do not carry out further disassembly if component is removed for access only.

  7. Remove 3x8 self tapping screws (x2) securing interior lamp to interior door release handle.
  8. Remove interior lamp from interior door release handle.

    Installation

  1. Installation is the reverse of removal procedure.

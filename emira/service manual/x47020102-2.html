Transmission Assembly - 4 Cylinder

Print

Operation Code: 47.02.01-02

Important Information

WARNING: Transmission oil is a toxic substance and can be lethal if ingested. Wear protective gloves, protective clothing and safety glasses.

    Removal

  1. Remove transmission assembly. Refer to procedure.
  2. Drain transmission oil. Refer to procedure.

    Remove Oil Pan

  1. Remove bolts (5) (x10) securing oil pan (1) to transmission.
  2. Remove oil pan.

    3. NOTE: Drain oil into a suitable container and dispose of appropriately.

  3. Remove and discard oil pan gasket (10).
  4. Remove and discard molded seals (11) (x2).
  5. Remove and discard plain couplers (12) (x2) with sealing rings.

    Remove Controller Unit Assembly

  1. Remove and discard M5x60 Torx bolts (5) (x15) securing controller unit assembly (Y3/14) to transmission (3).

    2. CAUTION: These fasteners have been torqued to a specification where the threads may have been damaged or stretched. Always discard and replace with new on installation.

  2. Remove controller unit assembly.
  3. Remove and discard sealing ring (8).

    Remove Dual Clutch

  1. Remove retaining ring securing dual clutch cover to transmission.

    2. NOTE: Do not damage seat of retaining ring and sealing surface on housing otherwise leaks may occur.

  2. Using a suitable tool, remove dual clutch cover.
  3. Remove shim.
  4. Remove axial bearing.
  5. Pull locking pin (A) upwards.
  6. Fully open lever (B).
  7. Install assembly tool (01) onto dual clutch (5).
  8. Pull locking pin upwards again.
  9. Close lever and latch locking pin into assembly tool.
  10. Remove dual clutch (5) from clutch housing (27K) using assembly tool.

    11. NOTE: Discard sealing rings (6).

    Install Dual Clutch

  1. Install dual clutch (5) into clutch housing (27K) carefully and without tilting using assembly tool.

    2. NOTE: Renew sealing rings (6).

    NOTE: Ensure sealing rings are correctly seated.

  2. Rotate dual clutch slightly until toothing of drive shaft (30) and hollow shaft (40) engages correctly in the dual clutch.
  3. Remove assembly tool from dual clutch.
  4. Measure installed dimension of dual clutch and adjust if necessary.
  5. Install axial bearing.

    6. NOTE: Inspect axial bearing for damage, renew if necessary.

  6. Install shim.
  7. Clean sealing surface on housing.

    8. WARNING: Only use cleaning agents and solvents in a well ventilated area.

  8. Install assembly sleeve onto drive shaft to protect sealing lip on cover.
  9. Install dual clutch cover.
  10. Install retaining ring.

    11. NOTE: Inspect retaining ring for damage, renew if necessary.

    NOTE: Ensure retaining ring is seated correctly.

  11. Remove assembly sleeve from drive shaft.

    Install Controller Unit Assembly.

  1. Renew sealing ring (8).
  2. Coat sealing ring with transmission oil.
  3. Install controller unit assembly (Y3/14).
  4. Install new M5x60 Torx bolts (5) (x15) securing controller unit assembly to transmission (3). Torque 3 Nm + 60 degrees.

    Install Oil Pan

  1. Clean component mating faces, prior to installation.

    2. WARNING: Only use cleaning agents and solvents in a well ventilated area.

  2. Install new plain couplers (12) (x2) with sealing rings.
  3. Install new discard molded seals (11) (x2).
  4. Install new oil pan gasket (10).
  5. Install oil pan (1).
  6. Install bolts (5) (x10) securing oil pan to transmission. Torque 10 Nm.

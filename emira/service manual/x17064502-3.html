Oxygen Sensor - Post Catalytic Converter - Left Side - V6

Print

Operation Code: 17.06.45-02

    Removal

  1. Remove rear undertray. Refer to procedure.
  2. Disconnect harness connector from oxygen sensor.
  3. Release clip securing harness connector to bracket.
  4. Release clip securing harness to heatshield bracket.
  5. Unscrew oxygen sensor from exhaust manifold. Torque 45 Nm.
  6. Remove oxygen sensor.

  1. Installation is the reverse of removal procedure except for the following:
  2. After installation perform a diagnostic read and clear error memory using Lotus Insight tool.

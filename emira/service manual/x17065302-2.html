Module - Differential Pressure Sensor (DPS) - 4 Cylinder

Print

Operation Code: 17.06.53-02

    Removal

  1. Disconnect battery. Refer to procedure.
  2. Remove engine cover. Refer to procedure.
  3. Disconnect harness connector from DPS module.
  4. Release clamps (x2) securing DPS module hoses to catalytic converter.
  5. Disconnect hoses (x2) from catalytic converter.
  6. Remove nut securing DPS module to bracket. Torque 8 Nm.
  7. Remove bolt securing DPS module to bracket. Torque 8 Nm.
  8. Remove DPS module.

    Installation

  1. Installation is the reverse of removal procedure except for the following:
  2. After installation perform a diagnostic read and clear error memory using Lotus Insight tool.

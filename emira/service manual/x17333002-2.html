Module - Key - 4 Cylinder

Print

Operation Code: 17.33.30-02

    Removal

  1. Remove rear quarter trim. Refer to procedure.
  2. Remove and discard cable ties (x2) securing key module to antenna.
  3. Remove key module.
  4. Release clip securing cover to key module.
  5. Open cover.
  6. Remove keyfob.
  7. Press button to release rear cover from keyfob.
  8. Remove cover.
  9. Pull out battery holder.
  10. Remove CR2032 coin cell battery.

    Installation

  1. Installation is the reverse of removal procedure except for the following:
  2. Install new CR2032 coin cell battery.
  3. Renew discarded cable ties.
  4. After installation perform a diagnostic read and clear error memory using Lotus Insight tool.

    5. NOTE: Register module using Lotus Insight tool.

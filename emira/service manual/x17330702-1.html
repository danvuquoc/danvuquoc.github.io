Module - Silent Alarm

Print

Operation Code: 17.33.07-02

    Removal

  1. Remove glove box assembly. Refer to procedure.
  2. Remove M5x12 bolts (x2) securing RH duct to vehicle. Torque 5 Nm.
  3. Release clips (x2) securing RH duct to inlet duct assembly.
  4. Remove RH duct.
  5. Disconnect harness connector from silent alarm module.
  6. Remove M4x16 Allen bolts (x2) securing silent alarm module to vehicle.
  7. Remove silent alarm module.

    Installation

  1. Installation is the reverse of removal procedure except for the following:
  2. Register silent alarm module with Scorpion.

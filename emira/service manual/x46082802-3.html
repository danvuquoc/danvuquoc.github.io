Thermostat - V6

Print

Operation Code: 46.08.28-02

    Removal

  1. Remove front engine bay trim cover. Refer to procedure.
  2. Release quick connector from thermostat.
  3. Disconnect oil hose from thermostat.

    4. NOTE: Be prepared to catch any excess fluid.

  4. Release clamps (x2) securing oil hoses to thermostat.
  5. Disconnect oil hoses (x2) from thermostat.

    6. NOTE: Be prepared to catch any excess fluid.

  6. Remove thermostat.

    Installation

  1. Installation is the reverse of removal procedure except for the following:

    2. CAUTION: Make sure quick connector clip is correctly secured between pipe and thermostat.

    NOTE: When installed correctly, retaining clip tangs should be visible in front of pipe flange. Refer to TSB 2023/10 for picture reference.

  2. Check and top up transmission oil. Refer to procedure.

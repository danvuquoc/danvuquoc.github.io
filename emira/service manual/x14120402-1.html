Latch - Glove Box

Print

Operation Code: 14.12.04-02

    Removal

  1. Remove glove box assembly. Refer to procedure.
  2. Open glove box lid.
  3. Remove screws (x3) securing glove box latch to glove box assembly.
  4. Remove glove box latch.

    Installation

  1. Installation is the reverse of removal procedure.

Coolant Reservoir

Print

Operation Code: 46.11.10-02

    Removal

  1. Remove airbox assembly. Refer to procedure.
  2. Remove coolant reservoir cap.
  3. Disengage hose clamp securing hose to coolant reservoir.
  4. Remove hose and allow coolant to drain.

    5. WARNING: Coolant is a toxic substance and can be lethal if ingested.

    NOTE: Drain coolant into a suitable container and dispose of appropriately.

  5. Disengage hose clamp securing hose to coolant reservoir.
  6. Remove hose.
  7. Remove M6 nuts (x2) securing coolant reservoir to vehicle. Torque 8 Nm.
  8. Remove coolant reservoir.

    Installation

  1. Installation is the reverse of removal procedure except for the following:
  2. Refill coolant reservoir.
  3. Start engine and allow to idle.

    4. NOTE: Run the engine with the reservoir cap off to bleed the system.

  4. Refit coolant reservoir cap.

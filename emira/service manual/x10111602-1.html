Tailgate Release Cable - Manual

Print

Operation Code: 10.11.16-02

    Removal

  1. Remove tailgate latch cover. Refer to procedure.
  2. Release straps (x2) and remove vehicle tool kit.
  3. Release strap and remove warning triangle.
  4. Remove luggage floor carpet.
  5. Release clips (x8) securing rear boot trim to boot box.
  6. Refer to illustration for trim clip location.
  7. Remove rear boot trim.
  8. Release clips (x2) and open tailgate release cable cover.
  9. Disconnect tailgate manual release cable from tailgate release cable.
  10. Release clips (x3) securing tailgate manual release cable to boot box.
  11. Remove rear left wheelarch liner. Refer to procedure.
  12. Release grommet and withdraw tailgate manual release cable from boot box
  13. Release clips (x2) securing tailgate manual release cable to vehicle.
  14. Remove rear left quarter trim. Refer to procedure.
  15. Release grommet and withdraw tailgate manual release cable from wheelarch.
  16. Release clips (x2) securing tailgate manual release cable to rear bulkhead.
  17. Release clip securing tailgate manual release cable to vehicle.
  18. Release tailgate manual release cable from bracket.
  19. Remove tailgate manual release cable.

    Installation

  1. Installation is the reverse of removal procedure.
